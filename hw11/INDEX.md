## 🚀 Быстрый старт (Docker - РЕКОМЕНДУЕТСЯ)

### ⚡ Вариант 1: Автоматическая установка (1 команда!)

```powershell
# Только для Windows с Docker Desktop
cd hw11
.\setup.ps1
```

**Что произойдет:**
- ✅ Установятся все зависимости (kubectl, k3d, argo CLI)
- ✅ Создастся локальный Kubernetes кластер (k3d)
- ✅ Установится Argo Workflows
- ✅ Применятся все 5 шаблонов
- ✅ Откроется UI на http://localhost:2746

**Время: 5-10 минут** (только первый раз)

📖 **[Прочитайте DOCKER_SETUP.md для защиты проекта!](DOCKER_SETUP.md)**

---

### 🔧 Вариант 2: Ручная установка (без Docker)

```bash
# 1️⃣ Установите Argo Workflows
kubectl create namespace argo
kubectl apply -n argo -f https://github.com/argoproj/argo-workflows/releases/download/v3.5.5/install.yaml

# 2️⃣ Примените шаблоны
kubectl apply -f workflow-templates.yaml -n argo

# 3️⃣ Запустите пример
kubectl create -f main-workflow.yaml -n argo
```

**Готово!** 🎉 Ваш первый workflow запущен.
